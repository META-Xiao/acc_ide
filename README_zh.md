## 项目功能与进度

### 核心IDE功能
- **代码编辑器集成**: 利用 `flutter_highlight` 进行语法高亮。
- **文件管理**: 包含带有文件列表、"新建文件"和"设置"选项的抽屉导航。
- **响应式编辑**: 集成了 `HighlightView` 和 `TextField` 以实现强大的代码编辑功能，支持多行和行号同步滚动。`TextField` 透明地叠放在 `HighlightView` 之上，实现无缝编辑。

### 自定义与用户体验
- **字体设置**: 支持自定义"Agave"字体，可调整字号和字重。
- **主题管理**: 允许用户选择各种代码主题（例如 'monokai'、'github'、'atomOneDark'）并在亮色和暗色应用模式之间切换。默认主题和模式由系统设置决定。
- **动态配色方案**: 应用程序的配色方案适应亮色和暗色模式，提供视觉愉悦且一致的体验。
- **侧边栏增强**: 侧边栏的标题部分包含自定义背景图片，并根据主题应用动态颜色滤镜，以及本地化的标题和副标题，提供个性化触感。

### 输入/输出面板
- **可拖动I/O面板**: 一个动态面板从屏幕底部向上滑出，初始高度为屏幕的40%，并可向上拖动至80%。
- **执行界面**: 包含"运行"按钮、状态显示（例如AC、WA、TLE等）以及"输入"、"当前输出"和"答案输出"的独立区域。
- **增强型UI**: 输入和输出字段具有圆角，外观现代，并改进了尺寸以提高可用性。

### 本地化
- **多语言支持**: 设置中包含语言切换开关，允许用户选择英语和中文。
- **全面本地化**: 所有关键UI元素，包括设置页面、新建文件对话框、I/O面板、侧边栏标题和副标题以及欢迎页面，都根据所选语言进行全面本地化。
- **系统语言默认**: 应用程序在首次启动时会智能地默认使用系统的首选语言。
