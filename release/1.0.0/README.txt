ACC IDE - Version 1.0.0
=============================

Release Date: June 19, 2025
Build Type: Release (Unsigned)

This package contains:
- acc_ide-1.0.0-unsigned.apk: Unsigned release APK


Installation Instructions:
1. For direct installation on developer devices:
   - Enable "Install from Unknown Sources" in device settings
   - Copy the APK to your device
   - Tap on the APK file to install

2. For signing and publishing:
   - Use Android's signing tools to sign this APK with your keystore
   - For Google Play, use the Play Console to upload the signed APK

Key Features in this release:
- Code editing with syntax highlighting
- File management system
- Light/Dark theme support
- Support for multiple programming languages
- Input/Output testing panel

Known Issues:
- Some deprecated methods are used and will be updated in future releases

For more information, please refer to the project documentation at:
https://github.com/META-Xiao/acc_ide
